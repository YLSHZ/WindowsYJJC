📁 硬件检测工具下载指南

1. 启动服务器：
   - 双击运行 "start_server.bat"
   - 服务器将在端口8000启动

2. 访问网站：
   - 在浏览器中打开：http://127.0.0.1:8000
   - 或者直接点击自动打开的浏览器窗口

3. 下载文件：
   - 点击 "硬件检测-完整显示版.bat" 旁边的 "下载" 按钮
   - 文件将被正确下载到您的电脑

💡 提示：
- 这种方式绕过了浏览器对本地文件的安全限制
- 所有文件都保存在 h:\ 目录下
- 服务器只在您需要时运行，关闭窗口即可停止
